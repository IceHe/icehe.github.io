Tools 利器

2017.03 | Mac |
我的 Mac Apps、电子产品推荐。
  • 主要介绍 macOS、iOS 的 Apps，简要提及 Windows。
  • Principles 原则
    • 选择工具，最重要的是「实用」，「美观」次之，「价格」不要太贵就行。
    • 「常用」的工具才值得折腾，使用机会少的工具就凑合着用吧，不要荒废大好时光。
    • 适合自己的工具，才是最好的工具。所以可以参考本文，进行尝试。
  • References 部分参考
    • 利器 - 创造者和他们的工具 以及使用的方式和原则。
    • 少数派 - 高质量 App 推荐媒体，关于 iOS、Mac、硬件。
    • MacTips - Mac 使用经验和技巧。
  • The item with the supscript Best  implies it is one of the best and so on.
  • 删除了一些原来有记录的 Apps，虽然我长时间用过它们，但还是觉得它们没有提及的价值（它们连弃用的理由也没必要说明）。

Software

Common

  • Usual
    • Evernote (Premium) Favorite
      云笔记，第二大脑（知识管理）。
      • 好记性不如烂笔头，而如今知识更新之快，纸笔已跟不上，于是笔记软件大放异彩。云端存储同步，不怕丢，便于检索，甚至搜索图片中的文字。总是死记硬背没有出路，不能被检索的知识毫无意义。
    • 2Do Favorite
      GTD 类事务管理。
      • 功能齐全、好用，用法可简可繁。快捷键完备，深得我这个键盘党的喜爱！推荐使用 iCloud Reminders 的 CalDAV 同步方式。
      • 一开始用 Doit.im 入门 GTD，虽然它的 Web 端的操作很极致高效，但客户端做得不尽人意。后来发现自己的生活很规律，无需复杂的管理，于是只用 Apple 自带的 Calendar。考虑过团队协作类的 App，如 Worktile，但团队协作的功能都用不上，个人管理的功能又不够简便易用。后来又用免费版 Wunderlist 了，任务管理的操作远不如 Web 端的 Doit.im 极致，但客户端做得不错，勉强满足个人需求。现在用 2Do。
    • 1Password
      密码管理。一（两）个密码管理所有其它密码，可自动填写网站的帐号和密码表单进行登录。
      • 以前一直用不惯，觉得用浏览器自带的自动登录和 iCloud 的 KeyChain Access 来记录管理帐号和密码足矣。工作之后经济独立了，更加注意保护自己的个人隐私和财产安全了，要管理相关的虚拟信息多而杂乱，还有部分需要额外的加密。所以不得不开始使用密码管理工具。
      • Other choices :
        • Dashlane : Excellent but expensive!
        • LastPass : Able to use for free.
    • 双拼输入方案！Favorite
      高效，易学！比五笔容易掌握得多，习得的性价比高，赶紧走出你的舒适区来试试双拼吧！
      • 《做少数派中的少数派：双拼输入快速入门》
      • 《选择输入法的哲学：兼论双拼的优缺点》
  • Web
    • Shawdowsocks Ladder
      • 首选的 PC 番羽土啬方案：
        Shadowsocks 服务 + Mac 的 ShadowsocksX-NG + 浏览器的 proxy 类插件：
        如 FireFox 的 FoxyProxy，Chrome 的 SwitchyOmega。
      • 备选方案：VPN + CHNRoute + Dnsmasq。
    • Chrome Most Usual
      用 Inspect Element 等 Develop Tools 做 Web 的开发、调试（远胜 FireFox 的半吊子 Firebug ）。
    • Firefox Favorite
      配合 VimFX 拓展，可在浏览器内 全局使用 Vim 键位 浏览网页、操作浏览器！键盘党神器。
      • Chrome 很好，但更耗电，影响到了 Mac 的续航；更重要的是类 VimFX 的 Vimium、cVim 等 Chrome 拓展无法在页面未加载完毕的情况下使用 Vim 键位，不够极致。
      • 不过 Firefox 冷启动较慢，界面不够 Safari、Chrome 漂亮。毕竟工具是拿来用的，最重要的是高效。
        （可惜我使用时会碰到一些奇怪的 bug）

Cmd Line

  • Project
    • Git Essential
      分布式代码版本管理系统。
  • Editor
    • Vim | Neovim God-like
      编辑器之神（ Emacs 则是神的编辑器）。
      • 服务端开发、运维经常要在远程服务器编辑文本，而 vi 是 *nix 标配，遂 vi* 自然是必备技能。
    • spf13-vim Essential
      一整套 Vim 配置方案。
      • 比起漫无休止地折腾配置，不如遵从实用主义：站在巨人的肩膀上，直接使用久经考验的的配置方案。
        （根本用不上其中的很多功能，而且高配的 MBP 用起来竟然卡！其实用熟 Vim 后，根据个人习惯在 .vimrc 写好的基本设置，就够用了。）
  • Shell
    • Zsh Powerful
      比 Bash 更强大、便捷、高效的 Shell！
      • 使用 oh-my-zsh 即可解决配置较为复杂的问题。可能你的电脑用 Zsh 比较慢，赶紧换 SSD 吧。
      • Fish (a shell for the 90s 😂 ) 虽然很好，但存在兼容性问题，有些 Bash 的指令需要改写才能运行在 Fish 上。
      • Ref : Comparison of Command Shells - Wikipedia
    • oh-my-zsh Efficient
      管理 Zsh 配置的开源框架，预打包了相关的主题、插件、配置。
      • 配置过程傻瓜化，一条安装指令就能让你畅快地享受 Zsh 的强大与高效！
    • tmux Powerful
      终端多路复用软件，即命令行中的 「桌面」、「分屏工具」。
      • 允许一个用户在一个终端窗口或一个远程终端会话中，使用多个终端会话。
  • Package
    • Homebrew Best
      macOS 的包管理器。就如 agt-get 之于 Ubuntu，yum 之于 RedHat、CentOS 的存在。
    • Homebrew Cask Great
      用命令行的方式安装、更新 Mac App，再也不用忍受 AppStore 缓慢的下载速度了！
      • 这样就可以用脚本来安装、更新 macOS 的 Apps 了。
        （用的机会少，其实不必安装。）
    • dotfiles
      各路大神的 dotfiles 配置。
      （ dotfiles 是什么？其实并不需要那么多工具包，有需要再折腾才是正道。）
    • YADR
      Yet Another Dotfile Repo，一个便捷的命令行配置方案，包括 Vim、Zsh、tmux、Git、RubyGems 以及 Alias 等各种配置。新手可以一试。

Apps

  • Develop
    • iTerm2 Best
      macOS 下的终端仿真机，是系统默认自带的 Terminal 的最佳替代 App。
    • Dash To the point
      查阅 API 参考文档、管理代码片段的工具。功能单一却精准。
      （我还是习惯去官网看文档，毕竟在浏览器可以用我的 Vim 式快捷键操作。）
  • IDE
    • PyCharm,  PhpStorm Master Piece
      Python、PHP 的最佳 IDE！现阶段类型推导做得最好，各方面可圈可点。配合 IdeaVim 插件可用 Vim 键位编辑，因而可少记很多相关的 IDE 快捷键。
      • JetBrains 出品的其它 IDE： IntelliJ IDEA、WebStorm、RubyMine、CLion 等。
  • Editors
    其实可以完全用 Vim 来编辑文本文件（在命令行中）。
    • Sublime Text Popular
      代码编辑器，我用其替代 TextEdit 作为默认的文本编辑器。Vintage 模式，可用 Vim 键位进行操作。冷启动也快如闪电。
    • Atom Modern
      相当现代的代码编辑器，体验一流，但是冷启动慢，不如 Sublime Text。
    • VS Code 潜力股
      简洁好用，只是 Vim 插件的体验不如 Atom，所以我放弃了。
    • Typora Elegant
      Markdown 编辑器，所见即所得。国人制作，完全免费，支持 Mac、Windows！
      （我很少需要预览 Markdown 内容的显示效果，所以用一般编辑器即可。）
  • Web
    • Charles Best
      网络封包分析（抓包）工具。如 Fiddler 之于 Windows。主要用于「应用层」的分析。
    • Wireshark Powerful
      网络封包分析工具。比 Charles 强大得多，但是也复杂得多。主要用于「网络层」的分析。
      （因为工作上没有使用的必要，所以我暂时用不上）
    • Postman Powerful
      APIs 开发、测试、归档的辅助工具。
    • Squel Pro Free
      管理 MySQL 数据库的 GUI 工具。
    • Transmit
      FTP 的 GUI 工具。
      （rsync 命令比 FTP 的效率高多了，用法也丰富。一般情况下用 netcat 命令传输文件也够用了。）
  • Others
    • Adobe Acrobat Reader DC
      一般情况下，不使用额外的 PDF 阅读软件，Mac 系统自带的 Preview 就够用了。
      • 但要在商务等正式场景下的 PDF 文件有高端用法，这是还是得 Adobe 家的软件来处理，例如签证申请文件、合同、保密文件等。
    • OpenEmu
      游戏机模拟器！可以模拟众多的旧时代的游戏主机！怀旧党福音。
      • 我又可以方便地玩我最喜欢的 NDS 游戏《最终幻想战略版 A2》啦！
        （少玩为妙。）
    • Parallel Desktop Best
      虚拟机软件，将 Windows、Ubuntu 等操作系统无缝贴合到 macOS 中去，使用流畅自然。
      （因为我不需要用 Windows 平台独占的那些软件，所以几乎用不上本软件。）

Assistant

  • Shortcuts & Keyboard
    • Manico Shortcuts
      为 macOS 设计快速启动、切换 App 的工具。
      • 我是键盘党，以前用 Windows 已惯用快捷键与手势迅速打开、切换程序，因此找到了它。
        （可用 Keyboard Maestro 代替。）
    • Keyboard Maestro Geek, Best
      限制你的只有你的想象力！高度自定义的效率工具，难以一语道尽玄机。
      • 可以尝试用它 定制系统全局 Vim 快捷键；可以 配合 Keymo 用键盘控制鼠标；可以替代 aText、TextExpander，用以拓展文本，提升打字效率；甚至完全替代 Manico，用以快速启动、切换 Apps，并贴合个人使用习惯为其添加进阶功能！
    • Karabiner Geek, Best
      强大、稳定的键位修改工具，预定义了大量奇巧的修改方案。键盘党神器！
      • 可自定义快捷键去启动、切换 App，打开文件和网址，快速输入关键词跳转到网站的搜索页，甚至运行命令行！
      • 发挥想象力，配合命令行和脚本（如 AppleScript）等，还能做到更多！例如，它也可以实现 Manico 的功能，不过得费些时间。
        （噩耗：macOS Sierra (10.12) 中关于键盘控制的代码大改，所以这代及以后的 macOS 就不能使用 Karabiner 了。除非 Karabiner 作者进行重大更新，然而工作量颇大，可用的新版本遥遥无期……）
    • Seil
      Karabiner 的补充，主要修改 Caps Lock、 Esc、Shift、Ctrl、Option 等按键的键位。
    • Key Codes
      键盘码查询。不用查表，直接击键，即得出其键盘码。
      （可用于 debug 以上软件关于键位和快捷键的修改）
  • Enhance
    • Amphetamine
      防止 Mac 休眠。比同类软件 Caffine、Owly 都更功能完备、好用。
      • 原因：Mac 每次休眠后都会重启软件，导致软件丢失部分上下文，于是我无法在完全一样的环境下持续工作，一定程度上打断了工作，降低了效率。
    • ClipMenu Efficient
      剪贴板管理工具。
      • 支持 URL、纯文本、RTF、图片、文件等各种格式，包括剪贴历史、排序、大小写转换、自定义文段的调出。
    • Moom Efficient, Best
      窗口布局软件，便捷地移动、缩放窗口，更好地利用桌面空间。
      • 有关文章：《说说 Mac 分屏的事(2) - Moom 的10个技巧》
    • Spectacle
      后来我用 Moom 形成了固定的套路，主要用左右分屏和全屏等布局，更简洁的 就能满足了。
    • Bartender 2 Clean Menubar
      收起或彻底隐藏毋需过多关注的菜单栏 App 图标。
    • HyperSwitcher Best
      窗口切换（关于 ⌘ ⇥, ⌘ `）增强的软件。
      • 系统自带的窗口切换不够好用，特别是切换属于同一应用的多个窗口：
        • 使用三指下滑的触摸板手势或快捷键 ^ ↓，可以显示当前应用的所有窗口，然后点击选择其中一个进行切换，但是不够快捷。
        • 使用快捷键 ⌘ ` 足够快捷，但是兼容性不佳，很多应用没办法靠它来切换同一应用。
        • HyperSwitcher 完美解决了这个问题，而且还对窗口切换的功能进行了进一步增强！例如，选择切换的窗口时，会显示各应用的缩略图。
    • CheatSheet Tips, for Freshman
      快捷键快速提醒工具。长按 Command 键，显示当前程序的快捷键列表，方便快速入门 Mac 各处的快捷键。
      （新手专用…）

Seldom

  • 不常用 / 鸡肋
    放在这里的有一些公认很好的 Apps，但我觉得它们不常用甚至鸡肋，且看我的理由。
  • Efficiency
    • Alfred 键盘党的神兵利器
      Spotlight 的增强版！关键字缩写的检索还是 Spotlight 精准；要用命令的话，也基本在终端中操作，没有 Alfred 之流的用武之地；要启动程序的话，有 Manico 的支持足矣。
      • 很多人把它说得神乎其神，其实我并不觉得有大用。看看操作流程：先输入一些字符，Alfred 智能检索猜出你要做的事，如打开程序、文本，或者运行脚本等；然后，选一个选项执行。
      • 我更喜欢敲快捷键「一步到位」告诉 Mac 要做的事！ 做好脚本或录制好一系列操作，主要用 AppleScript、Keyboard Maestro 和 Automator 等，然后用快捷键激它们，真正常用的组合操作并不会很多。
    • BetterTouchTool
      为 Mac 的触摸板定制更多手势操作。iOS 版 App 还可以用来让 iPhone、iPad 设备化身为 Mac 的触摸板。现已是收费软件，但不贵。
      • 可是作为键盘党，自定义的快捷键足以让我驰骋 macOS，基本没机会用它。
    • Contexts Great
      App 窗口切换工具。操作行云流水，切换方面比 Manico 更优秀！但不能定制用于快速启动 App 的快捷键，因而被 Manico 替代。
    • iHosts
      灵活管理 macOS 的 hosts 的方案。
      • 可惜通常情况下，hosts 并不需要复杂的管理，还有很多其它手段达到同样的目的。
        不如直接在命令行加一个简短的 alias 用于直接打开修改 hosts 文件，这样比较省事。
    • MacID 不推荐
      可用其它 Apple 设备唤醒、解锁 Mac、甚至授权「提权」操作。
      • iPhone、iPad、AppleWatch 用蓝牙链接 Mac 后，可以用 TouchID、iOS 密码、设备的靠近和远离（距离）、蓝牙连接状态或触摸板手势等，来唤醒、解锁 Mac，或锁屏、提权。当指定便携设备远离 Mac，Mac 就会自动锁屏并发出提示；重新靠近 Mac 时，Mac 自动唤醒解锁！
      • 但蓝牙不时断连后，并不总能快速重连成功，比较恼人。新一代的 macOS 也将会原生支持 Apple 设备解锁 Mac。所以弃用，其它同类型的软件也半斤八两，均不推荐使用。
      • macOS Sierra (10.12) 之后，就可以使用 AppleWatch 解锁了；新版 MacBook Pro 也可以用指纹解锁了。所以 MacID 这类软件成了彻底的鸡肋！
    • Keymo 不推荐
      使用键盘操作鼠标，四象限定位法颇有创意！
      • 作为键盘党的我，真要用到鼠标的时候，还是用触控板比较舒心。我的触摸板、鼠标的触控灵敏度（移动速度）向来调到最高，操作还是很快的。
      • 这种操作方式不如直接用鼠标那样符合人的直觉，需要不少练习，所以不值得使用。
    • Shortcat 不推荐
      UI 上的 Spotlight，暂时难堪大用…… 不如快捷键来得直接。
  • Enhance
    • Quick Look plugins
      在 Finder 浏览时，选中文件，再按空格键便可以使用 macOS 提供的简单预览。
      • 该 Github 仓库提供针对各种类型文件的更强大的预览功能的插件。是否有大用见仁见智，对我来说有点鸡肋，留着也无碍。
    • Default Folder X
    • Path Finder 不推荐
      用于替代、增强系统默认的文件管理器 Finder。
      • 我通常都用终端命令行去管理文件系统，Finder 的快捷键烂熟于心，没必要再加额外的工具帮助。
    • Boom 不推荐
      音效增强软件。可用于突破系统音量的上限，留意音量过大可能损坏 Mac 音箱。可有可无。
    • HazeOver Focus
      瞬间找到当前正在使用窗口。除当前激活的窗口，其它窗口都会显示阴影，让人更专注与当前窗口的工作。
      • 可惜经过长期使用，感觉可有可无，上述说法可能只有理论上的作用和心理安慰的作用。
    • HyperDock
      窗口增强工具。光标停到 Dock 的 App 图标上，能快速预览该软件的所有窗口，点击切换到不同的窗口或桌面，还有快速调整窗口布局、大小、位置等的功能。
      • 用处还是不大，其实只要用 HyperSwitch 去切换一个 App 各个子窗口然后直接查看就够了。
    • iStat Menus
      支持顶部菜单栏部分自定义，显示硬件状态的监控软件。方便查看 CPU、内存、网络、硬盘及其它硬件的状态，还有日期时间栏的定制。
      • Mac 没异常时，知道这些状态并没有显著的用处，如果设备崩溃了、卡了，还不是直接重启了事。
    • ShowyEdge
      输入法状态提示。显示一条样式可定制的色带在屏幕顶部，以提示正在使用什么输入法，在全屏使用 App 时，也能轻易区分输入法状态！
      • 我的键盘设置了一键直接切换输入法，简单粗暴。输入法状态了然于心，用不着它了。
    • f.lux Health
      根据所在地当时的日照，自动调整屏幕色温、亮度，减少蓝光对眼睛的刺激，以调整激素水平，保护正常睡眠节律。
      • 然而效果相同的 Night Shift 功能已经在 10.12.4 版本整合到了 macOS 中！

  • Headphone
    • Bose QC30 wish!
      无线蓝牙连接，充电 <3h，续航 10h；消噪程度可控，分多级。
      续航时间比 QC20 短，不能像 QC20 那样一键切换「消噪」和「接收外部声音」两个状态，比较麻烦。
    • Bose QC20 Favorite, Best
      QuietComfort® 20 Acoustic Noise Cancelling® headphones — Apple devices
      • 目前市面上我所知的最好的降噪耳塞。主动降噪，耳塞发出一种波与有规律的噪音相消。
      • 可「一键切换」是否隔绝周围环境的噪音；降噪后，可用更小的音量使用耳塞，声音更清晰。
      • 人声不能完全隔绝，但其音量会变得很小；主动降噪时，再稍微播放音乐或白噪音，即可在很大程度上隔绝外部噪音。它帮助我在嘈杂的办公室也能在静谧、专注地做事。虽然它贵，但花得值！

    • Sony SRS-X1-V Favorite
