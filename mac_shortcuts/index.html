Explanation

Not all shortcuts , but also my most frequent ones!
Not only the default shortcuts, but alos the custom ones.

Key Characters

 Power
⌘ Command ( Cmd )
⌥ Option | Alt ( on Windows )
^ Control ( Ctrl )
⇧ Shift ( Shf )
⇪ CapsLock
⇥ Tab
⎋ Escape ( Esc )
↩ Return ( Ret ) | Enter ( on Windows )
凵 Space

↑ Up
↓ Down ( Dn )
← Left ( Lf )
→ Right ( Rg )

⌫ Delete ( on Mac ) | Backspace ( on Windows )
Num Digits: 0 ~ 9

Abbreviations

  • Button

    Cmd Command
    Ctrl Control
    Esc Escape
    Opt Option
    Ret Return
    Shf Shift
    Del Delete

  • Direction & Position

    Dn Down
    Lf left
    Rg Right

    Prev Previous
    Btm Bottom

  • Word

    App Application
    Btn Button
    Dir Directory

    Mv Move
    Rm Remove

    Ld Leader
    Pf Prefix

  • Symbol

    & And
    | Or

Examples

  • ⌘ ⌥ a Manipulation

    Press “Cmd”, “Opt” & “a” at the same time to do the manipulation .

  • A Manipulation

    The capital letter “A” means pressing “Shf” & “a” at the same time !

  • Shortcut1 | Shortcut2 Manipulation

    The manipulation has two shortcuts Shortcut1 & Shortcut2 .

  • ⌘ [ 1 | 2 ] Manipulation

    Press “⌘ 1” or “⌘ 2” to do the manipulation .

Meaning of Font Style

Bold : I am unfamiliar with it .
Italic : I seldom use it .

tmux

^ a Prefix Key

  • After inputing Prefix Key ^ a

    • Help

      Pf ⇧ ? List Keys

    • Client (Session)

      Pf ( Prev Client
      Pf ) Next Client

      Pf d Detach Client
      Pf z Kill Client

      Pf $ Rename Session

    • Window

      Pf c New Window
      Pf & Kill Window

      Pf w Choose Window
      Pf p Prev Window
      Pf n Next Window

    • Layout

      Pf \ Split Window Horizontally
      Pf - Split Window Vertically

      Pf 凵 Another Layout
      Pf ^ o Rotate Window

      Pf { Swap Pane Up
      Pf } Swap Pane Dn

      • Resize

        Pf ^ h Resize Pane Lf
        Pf ^ j Resize Pane Dn
        Pf ^ k Resize Pane Up
        Pf ^ l Resize Pane Rg

    • Pane

      Pf x Kill Pane

      Pf h Select Lf Pane
      Pf j Select Dn Pane

      Pf l Select Rg Pane

      Pf ↓ Put Current Pane back to its Parent Window

    • Copy Mode

      Pf r Reload ~/.tmux.conf

Vim

Only List the Useful Keys that I’m unfamiliar with.

  • Del

    x Del Char Forward ⌦
    ⇧ X Del Char Backward ⌫

    s Del Char Forward & then Insert
    ⇧ S Del Current Line & then Insert
    ⇧ C Del to End of Line & then Insert
    ⇧ D Del to End of Line

  • Selcet Range

    ciw Del Word
    caw Del Word including the Following Spaces 凵

    ci* Select & Manipulate the string surrounded by *.
    ca* Select & Manipulate the string surrounded by * including *.

  • Cp

    ⇧ Y Copy Line without Line Break ‘\n’ | ‘\r’

  • Exchange

    xp Exchange the Current Char and the Next Char
    ddp Exchange the Current Line and the Next Line

  • Check Encoding Value

    ga Show ASCII of Char
    g8 Goto UTF-8 of Char

  • Open Path

    gf Open Path where Cursor is

  • Lower or Upper Case

    u to Lowercase
    ⇧ U to Uppercase

  • Increase or Decrese Num

    ^ a Increase Num
    ^ x Decrease Num

  • Mv Cursor

    ge Goto Last Char of Prev Word

    m Set Mark
    ' Jump to Mark
    ` Jump to Mark

    ⇧ } Next Paragraph

    ⇧ + Head of Next Line
